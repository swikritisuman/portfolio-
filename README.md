# 💼 Swikriti Suman – Personal Portfolio

A modern, interactive personal portfolio built with HTML, CSS, and JavaScript. 
Features AI/ML project showcases, voice welcome, animated UI, and a clean dark theme.

## 🚀 Live Demo
[View Portfolio](https://swikritisumanportfolio.netlify.app/)

---

## 🧰 Tech Stack

| Layer | Technologies |
|-------|-------------|
| Markup | HTML5 |
| Styling | CSS3, Flexbox, Grid, Custom Animations |
| Logic | Vanilla JavaScript |
| Icons | Font Awesome 6 |
| Fonts | Inter, Syne (Google Fonts) |
| Forms | Formspree |
| Voice | Web Speech Synthesis API |

---

## ✨ Features

- 🎬 **Splash screen** with animated glowing logo
- ⌨️ **Typewriter effect** cycling through role titles
- 🔊 **Voice welcome message** with mute/unmute toggle
- 🌟 **Falling star animation** across the background
- 📱 **Fully responsive** across all screen sizes
- 🧭 **Active nav highlight** on scroll
- 📎 **Floating resume button** for quick CV download
- 💬 **Contact form** with AJAX submission via Formspree
- 🃏 **Project cards** with hover effects and GitHub links
- 🏆 **Research & Achievements** sections with award highlights
- 🌙 **Dark theme** with cyan + purple accent palette
